## Claude Code 专属指令

### 交互偏好

- 使用中文进行所有对话和代码注释
- 回答简洁直接，避免冗长的解释
- 在执行复杂操作前先询问确认
- 代码变更后主动展示关键改动

### 工具使用

- 优先使用 Read/Grep/Glob 等工具收集信息，减少不必要的 Shell 命令
- 对于需要执行的操作，明确告知用户将要做什么
- 涉及到代码修改的操作，需要用户明确确认后再执行

### 开发流程

- 先理解需求，再开始编码
- 复杂功能先讨论方案再实现
- 提交代码前检查是否有遗漏

### 禁止事项

- 不要在未经用户确认的情况下执行破坏性操作（如删除文件、清空目录）
- 不要修改 `nginx-1.20.2/` 目录下的前端代码（已打包完成）
- 涉及数据库结构的变更脚本，必须放入 `sql/` 目录管理
